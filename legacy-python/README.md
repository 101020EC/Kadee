# Buntuek - ระบบกรอกบันทึกข้อความศุลกากรอัตโนมัติ

แอปพลิเคชันเว็บที่ช่วยอำนวยความสะดวกในการอ่านไฟล์ PDF ใบขนสินค้าพิเศษ และสกัดข้อมูลเพื่อไปกรอกลงในบันทึกข้อความราชการของกรมศุลกากร (ไฟล์ Word) โดยอัตโนมัติ

## โครงสร้างโปรเจกต์
- [MY.docx](file:///Users/karn/Documents/Developer/Antigravity/Buntuek/MY.docx): ไฟล์บันทึกข้อความเดิมที่ได้รับการปรับปรุงแล้ว โดยแทนที่ข้อความไฮไลท์สีเหลืองทั้งหมดด้วยตัวแปรเทมเพลต (เช่น `{{case_number}}`, `{{person_name}}` เป็นต้น) และลบไฮไลท์สีเหลืองออกเรียบร้อยแล้ว
- [app.py](file:///Users/karn/Documents/Developer/Antigravity/Buntuek/app.py): เซิร์ฟเวอร์ Flask ที่ทำหน้าที่ประมวลผล PDF, ดึงค่าต่างๆ, แปลงวันที่ของคริสตศักราช (A.D.) เป็นพุทธศักราช (พ.ศ.) และสร้างไฟล์ Word ส่งคืนให้ผู้ใช้ดาวน์โหลด
- [templates/index.html](file:///Users/karn/Documents/Developer/Antigravity/Buntuek/templates/index.html): หน้าเว็บอินเตอร์เฟสแบบ Glassmorphism โทนสีมืดที่สวยงาม รองรับการลากและวางไฟล์ (Drag-and-Drop) และการตรวจสอบ/แก้ไขข้อมูลก่อนส่งออก
- [8479.pdf](file:///Users/karn/Documents/Developer/Antigravity/Buntuek/8479.pdf): ไฟล์ตัวอย่างสำหรับใช้ทดสอบการอัปโหลด

## รายชื่อตัวแปรที่ใช้แทนที่ไฮไลท์ (Template Variables)
1. `{{case_number}}`: เลขแฟ้มคดีหลัง ดปบ.ร. (เช่น `926/18.6.69`)
2. `{{customs_office}}`: ชื่อด่านศุลกากร (เช่น `ด่านศุลกากรปาดังเบซาร์`)
3. `{{declaration_number}}`: เลขที่ใบขนสินค้าพิเศษ (เช่น `5901-7690504405`)
4. `{{person_name}}`: ชื่อผู้นำเข้าชั่วคราว/ผู้ขออนุญาต (เช่น `MRS. MUSALMAH BINTI MD ARIP`)
5. `{{vehicle_type}}`: ประเภทของยานพาหนะ (เช่น `รถจักรยานยนต์`)
6. `{{vehicle_brand}}`: ยี่ห้อของยานพาหนะ (เช่น `HONDA`)
7. `{{vehicle_plate}}`: ทะเบียนรถ (เช่น `RBD 8479`)
8. `{{import_date_th}}`: วันที่นำเข้ามาในราชอาณาจักร (เช่น `18 พฤษภาคม 2569`)
9. `{{due_date_th}}`: วันที่ครบกำหนดสัญญาประกัน (เช่น `16 มิถุนายน 2569`)
10. `{{return_date_th}}`: วันที่นำรถกลับออกไปจริง (เช่น `11 มิถุนายน 2569`)
11. `{{doc_date_th}}`: วันที่จัดทำบันทึกข้อความ / วันที่ตัดบัญชี (เช่น `18 มิถุนายน 2569`)
12. `{{dept_abbr}}`: อักษรย่อฝ่ายรับผิดชอบ (เช่น `ฝคต`)
13. `{{proposer_name}}`: ชื่อเจ้าหน้าที่ผู้เสนอรายงาน (เช่น `นายสุรวัฒน์ โชติช่วง`)
14. `{{proposer_position}}`: ตำแหน่งเจ้าหน้าที่ผู้เสนอรายงาน (เช่น `นักวิชาการศุลกากรชำนาญการ`)
15. `{{approver_name}}`: ชื่อผู้อนุมัติ/ผู้เปรียบเทียบปรับ (เช่น `นายวรวุฒิ สุภชัยพานิชพงศ์`)
16. `{{approver_position}}`: ตำแหน่งผู้อนุมัติ (เช่น `หัวหน้าฝ่ายควบคุมและตรวจสอบทางศุลกากร`)

## วิธีการติดตั้งและเริ่มใช้งาน

### 1. ติดตั้งไลบรารีที่จำเป็น
เปิด Terminal ในโฟลเดอร์ของโปรเจกต์นี้ และรันคำสั่งติดตั้งไลบรารี:
```bash
pip install flask python-docx pypdf
```

### 2. เริ่มรันแอปพลิเคชัน
รันแอปพลิเคชันเซิร์ฟเวอร์ด้วยคำสั่ง:
```bash
python app.py
```

### 3. เปิดใช้งานผ่านเบราว์เซอร์
เมื่อระบบเริ่มรันแล้ว ให้เปิดเว็บเบราว์เซอร์และเข้าไปที่ URL ด้านล่างนี้:
```text
http://localhost:8080
```

### 4. วิธีการทำงาน
1. ลากไฟล์ PDF ใบขนสินค้าพิเศษของคุณ (เช่น [8479.pdf](file:///Users/karn/Documents/Developer/Antigravity/Buntuek/8479.pdf)) ไปวางในกล่องอัปโหลด
2. ระบบจะอ่านข้อมูล ประมวลผล และสกัดฟิลด์ต่างๆ ขึ้นมาแสดงในฟอร์มเพื่อให้ตรวจสอบความถูกต้องโดยอัตโนมัติ
3. คุณสามารถแก้ไขข้อมูลในฟิลด์ที่ต้องการ หรือกรอกข้อมูลเพิ่มเติม (เช่น วันที่นำออกจริง หรือรายละเอียดเจ้าหน้าที่)
4. คลิกปุ่ม **"สร้างบันทึกข้อความ Word"** เพื่อรับไฟล์ `.docx` สำเร็จรูปที่ได้รับการกรอกข้อมูลและลบสีไฮไลท์เรียบร้อยแล้วไปใช้งานได้ทันที
